# crud-pessoas-api


Desafio Java
Funcionalidades da Api em cada EndPoint:
 Criar uma pessoa
 Editar uma pessoa
 Consultar uma pessoa
 Listar pessoas
 Criar endereço para pessoa
 Listar endereços da pessoa
 Poder informar qual endereço é o principal da pessoa

Modelo de Pessoa:
 Nome
 Data de nascimento
 Endereço:
o Logradouro
o CEP

o Número
o Cidade
